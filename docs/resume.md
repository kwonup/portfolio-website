# Projects

## 토킷(Tokit)

**IT취업준비생을 위한 커피챗 네트워킹 플랫폼**  
**2025.07 - 2025.09**

- 링크: https://github.com/lionchat100/lion-chat-fe

Tokit은 IT 업계 종사자와 취업 준비생을 위한 네트워킹 플랫폼으로, 사용자가 프로필을 등록하고 기술 스택, 관심사 기반으로 연결될 수 있도록 돕는 서비스입니다.  
배포 후 해커톤 사용자 포함 실제 가입자가 1,500여 명에 달했고, 사용자 피드백을 수집해 UI 기능 개선에 반영한 경험이 있습니다.

Next.js, TypeScript, TailwindCSS 기반으로 개발했으며 기획·디자인에도 참여했습니다.  
프론트엔드 주요 담당 업무는 다음과 같습니다.

- 홈 화면
- 프로필 탐색
- 프로필 상세
- 마이페이지

또한 FSD 아키텍처를 적용해 유지보수성과 확장성을 고려한 구조로 구현했습니다.  
Claude Code CLI를 팀 워크플로우에 도입해 반복적인 코드 생성, 리팩토링, 디버깅 플로우를 효율화했고, 작업 단위를 컴포넌트/로직 단위로 잘게 나누는 방식으로 개발 사이클을 단축했습니다.

## 머핀(Must-Fin)

**투자 입문자들을 위한 금융교육 및 모의투자 서비스**  
**2025.02 - 2025.03**

- 링크: https://github.com/FRONT-END-BOOTCAMP-PLUS-3/mufin

머핀은 투자 입문자들이 금융과 투자에 보다 쉽게 다가갈 수 있도록 돕기 위해 기획된 서비스입니다.  
사용자는 금융 및 투자 상식과 관련된 퀴즈를 풀고 이에 대한 보상으로 포인트를 획득하며, 이를 활용해 모의투자를 진행할 수 있습니다.  
이를 통해 자연스럽게 금융 지식을 학습하고 투자 경험을 쌓을 수 있도록 설계했습니다.

Next.js, TypeScript, Prisma, styled-components를 기반으로 개발했으며,  
클린 아키텍처를 적용해 도메인 로직과 외부 요소를 분리함으로써 유지보수성과 확장성을 고려했습니다.

주요 구현 내용은 다음과 같습니다.

- 모의투자 메인 페이지
- 보유자산(지갑) 페이지
- 검색 페이지

또한 Chart.js를 활용해 주식 차트를 시각화했고, Socket 통신을 통해 실시간 주식 데이터를 받아오는 기능을 구현했습니다.  
또한 한국투자증권 API를 활용해 실제 API를 호출하여 실시간 데이터를 기반으로 한 모의투자 서비스를 구성했습니다.

## 짐넥트(Gym-Nect)

**운동을 기록하고 서로 공유하는 운동 아카이빙 SNS 플랫폼**  
**2023.10 - 2023.11**

- 링크: https://github.com/FRONTENDSCHOOL7/final-07-gymnect

짐넥트는 사용자들이 웨이트 트레이닝, 러닝 등 다양한 운동 루틴을 기록하고 공유하며,  
일주일 동안의 운동 성과를 시각적으로 확인할 수 있도록 설계된 SNS 서비스입니다.  
웨이트는 무게와 횟수, 러닝은 시간과 거리 등을 기준으로 기록되며,  
운동 성과를 친구들과 공유하며 서로에게 긍정적인 자극과 동기부여를 줄 수 있도록 하는 데 중점을 두었습니다.

React, Recoil, Axios, styled-components 기반으로 개발했으며,  
프로젝트 초기 기획과 아이디어 도출에 참여했습니다.

주요 기능은 다음과 같습니다.

- 메인 페이지
- 로그인
- 검색 기능

또한 러닝과 웨이트 기록 상세 페이지에서는 Chart.js를 활용해 운동량을 그래프로 시각화했고,  
캘린더 UI와 연동하여 주간 또는 월간 운동량을 한눈에 확인할 수 있도록 구현했습니다.  
로그인 상태 관리를 위해 Recoil을 활용했으며, React Router를 통해 권한 기반 라우팅을 적용하여 인증 사용자만 접근할 수 있도록 보안성을 높였습니다.  
또한 비만은 기입을 활용해 검색 기능을 통해 성능을 개선했고, 팔로우한 사용자의 피드를 동적으로 렌더링하는 기능을 개발해 사용자 경험을 강화했습니다.

## EF(Emotion Feedback)

**AI 감정분석 기반, 대화 주제 추천 화상 소개팅 서비스**  
**2024.03 - 2024.06**

- 링크: https://github.com/EmotionFeedback

화상통화를 통해 소개팅 상대를 연결해주고, 통화 중 표정과 음성 데이터를 활용한 AI 감정 분석을 통해 사용자의 감정 상태를 수치화 및 시각화하는 기능을 제공하는 화상 소개팅 서비스입니다.  
소개팅 종료 후에는 두 사람의 감정 흐름 그래프를 보여주어 대화의 분위기를 되짚어볼 수 있도록 했으며,  
대화 중 할 말이 없어졌을 때는 실시간으로 대화 주제를 추천해 자연스럽고 끊김 없는 소개팅이 이뤄지도록 돕는 데 초점을 맞춘 서비스입니다.

JavaScript, React, Recoil, WebSocket, WebRTC를 기반으로 개발했으며,  
프로젝트 기획과 아이디어 선정부터 UI 고안까지 전반을 담당했습니다.

주요 기능은 다음과 같습니다.

- WebRTC와 WebSocket을 활용한 실시간 화상채팅 기능
- Recoil을 통한 상태 관리
- 음성 및 음소 스트림 제어
- 대화 주제 추천 API 연동
- 사용자 감정 시각화 기능
- 얼굴 표정 데이터 및 음성 데이터 서버 업로드 기능

감정 분석 및 피드백 시스템 구현 경험이 있습니다.
